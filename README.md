# SHIB

SHIB is my streamlined Wave-based terminal build with a tighter UI, easier copy workflows, and a cleaner day-to-day setup.

## What Changed

- `SHIB` branding in the packaged app and visible window chrome
- Wave AI input area tuned with extra right-side breathing room
- Terminal text selection made more readable while dragging to copy
- Faster terminal copy UX with simpler drag-to-copy behavior

## Local Development

```bash
npm install
npm run dev
```

Build a packaged app:

```bash
npm run build
```

## Notes

- This repo is based on Wave / Wave Terminal upstream.
- Internal compatibility names and lower-level runtime wiring are intentionally left mostly intact where changing them would add migration risk.
